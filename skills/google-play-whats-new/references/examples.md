# Examples

## Example prompts

- `Use $google-play-whats-new to write a 500-character English What's New for Gallery v2.8 from this PRD.`
- `Use $google-play-whats-new to tighten this Google Play release note to under 300 characters.`
- `Use $google-play-whats-new to compare my draft with the source DOCX and fix unsupported wording.`

## Example output pattern

Recommended:

`V2.8 – What's New`

`Introducing Draw in Gallery. Add doodles, notes, and highlights to your photos with customizable colors, stroke sizes, and opacity. Need to hide sensitive details? Use Mosaic to blur private areas quickly. You can also add shapes like lines, arrows, rectangles, and circles, then select them to edit, rotate, stretch, or delete.`

`Characters: 382`

## Shortening strategy

When the copy is too long:

1. Remove version headers first if the user does not require them.
2. Compress benefit phrases before removing feature coverage.
3. Replace long noun phrases with simpler verbs.
4. Keep the top `2` or `3` user-visible changes.
